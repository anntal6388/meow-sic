# 🐱🎵 Meow-sic

A fun cat piano app where you click on cute cats to play Do Re Mi Fa So — in piano or cat meow mode!

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Framework: Vite
4. Build command: `npm run build`
5. Output directory: `dist`

That's it! 🐾
